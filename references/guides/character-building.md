# 人物塑造指南

## 人物模板

```markdown
## 人物名

**基本信息**
- 年龄：
- 身份：
- 外貌特征：（2-3个关键特征即可）

**核心动机**
- 他最想要什么？
- 为什么想要？（背后的原因）
- 愿意为此付出什么代价？

**核心缺陷**
- 阻止他得到目标的内在障碍是什么？
- 这个缺陷在故事中如何影响他？

**人物弧线**
- 故事开始时他是什么状态？
- 故事结束时他变成什么状态？
- 关键转变发生在哪里？

**关系网络**
- 与主角：{关系描述}
- 与反派：{关系描述}
- 与其他角色：{关系描述}
```

## 人物设计要求

### 好的设定
- 有明确欲望（不是"想变强"，而是"想保护妹妹"）
- 有内在矛盾（勇敢但冲动 / 聪明但自负）
- 有成长空间（开局不完美）
- 有不可替代性（换一个人物故事就不成立）

### 坏的通病
- 完美无缺（没有成长空间）
- 工具人（只为主角服务）
- 标签化（高冷/腹黑/呆萌，用完即弃）
- 静态无变化（从头到尾一个样）

## 人物关系网
- 画三角形关系：A爱B，B爱C，C恨A
- 每个关系都有张力，不是"好朋友"那么简单
- 利益绑定比感情绑定更牢固
- 关系变化是情节推进的关键信号

## 常用人物弧线类型
1. **成长型**：自卑→自信
2. **堕落型**：正直→堕落
3. **觉醒型**：懵懂→认清真相
4. **救赎型**：有罪→赎罪→解脱
5. **坚守型**：面临诱惑→守住底线
