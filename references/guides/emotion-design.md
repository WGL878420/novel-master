# 情绪设计指南

整合自 oh-story 的情绪设计模块

## 六种情绪弧线

### 1. 希望→挫折→希望
适用于：升级流、奋斗型故事
```
开场: 主角看到希望
中段: 遭遇挫折，希望破灭
结尾: 新的希望出现（更强）
```

### 2. 甜蜜→误解→虐心→和解
适用于：言情、感情线
```
开场: 甜蜜互动
中段: 误会产生，关系恶化
高潮: 最虐心的时刻
结尾: 真相揭示和解
```

### 3. 平静→危机→紧张→解决
适用于：悬疑、惊悚
```
开场: 日常平静
中段: 危机出现，紧张升级
高潮: 最大危机
结尾: 危机暂时解决（或留下新悬念）
```

### 4. 压抑→小爽→更压抑→大爽
适用于：爽文、逆袭
```
开场: 主角被压制
中段: 小规模反抗成功
高潮: 更强大的压制
结尾: 大爽（阶段性胜利）
```

### 5. 未知→探索→震惊→领悟
适用于：解密、悬疑
```
开场: 面对未知
中段: 逐步探索发现
高潮: 震惊性真相
结尾: 更深层次的理解
```

### 6. 轻松→紧张→悲伤→治愈
适用于：生活流、治愈系
```
开场: 轻松日常
中段: 紧张事件发生
高潮: 悲伤时刻
结尾: 温暖和解/治愈
```

## 节奏控制原则

### 情绪起伏
- 情绪高点后需要低点缓冲（不能一直高潮）
- 情绪低点后需要上升空间（不能一直压抑）
- 高低交替周期：2-3章一个循环

### 每章情绪基调
- 不要在情绪高潮章加大量设定说明
- 不要在情绪低落章安排搞笑情节
- 保持每章情绪基调统一

### 情绪强度递进
- 不要第3章就用到最强冲突
- 保留20%的冲突强度给最后的1/3
- 每卷的情绪高潮要强于前一卷

## 情感节点分布

### 每卷建议
| 卷位置 | 情感任务 |
|--------|---------|
| 卷一 | 建立共情（让读者在意主角） |
| 卷二 | 深化投入（让读者在乎关系/目标） |
| 卷三 | 高 emotional stakes（让读者紧张结果） |
| 卷四+ | 持续施压+阶段性释放 |

### 情感节奏检查
- [ ] 最近3章情绪是否太单调？
- [ ] 读者有喘息的空间吗？
- [ ] 关键时刻的情绪冲击力够吗？
- [ ] 情绪转换是否自然（不突兀）？
