# 情节结构模板

## 三幕式结构（通用，推荐）

### 第一幕：设置（约25%）
| 章节 | 内容 |
|------|------|
| 第1章 | 介绍主角日常生活/异常开局 |
| 第2章 | 激励事件——打破现状 |
| 第3章 | 主角决定行动 |

### 第二幕：对抗（约50%）
| 章节 | 内容 |
|------|------|
| 第4-5章 | 进入新世界，遭遇挑战 |
| 第6章 | 中点——重大转折/信息揭示 |
| 第7-8章 | 困难升级，盟友或背叛 |

### 第三幕：结局（约25%）
| 章节 | 内容 |
|------|------|
| 第9章 | 看似失败，最低点 |
| 第10章 | 高潮——最终对决 |
| 终章 | 结局，展示新常态 |

## 英雄之旅（冒险/奇幻）
12阶段：平凡世界→冒险召唤→拒绝召唤→遇见导师→跨越门槛→考验盟友→接近洞穴→苦难煎熬→奖赏→返回之路→复活→满载而归

## 悬疑结构（侦探/推理）
第一幕：谜题出现 → 第二幕：调查深入（审问/新线索/第二事件） → 第三幕：真相揭示（反转/对决/遗留疑问）

## 言情结构
第一幕：相遇（第一印象不佳）→ 第二幕：发展（被迫相处→吸引→障碍→危机）→ 第三幕：和解（真爱→克服→圆满）

## 多线叙事
- 交叉剪辑：各主角交替推进，最终交汇
- 时间线：现在/过去交替，逐步揭示真相

## 章节号-情节对照速查
| 结构\章数 | 10章 | 20章 | 30章 |
|-----------|------|------|------|
| 第一幕 | 1-3 | 1-5 | 1-8 |
| 第二幕 | 4-8 | 6-15 | 9-22 |
| 第三幕 | 9-10 | 16-20 | 23-30 |
