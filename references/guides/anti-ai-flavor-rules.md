# 去AI味完整操作手册

## 核心原则（三句话记住）

- 能用动作不用总结
- 能用对白不用解释
- 能写具体不写抽象

---

# 第一层：初级规则（10分钟速查）

## 一、高频AI词汇禁用表

| 禁用词 | 替换方案 |
|--------|---------|
| 不禁/不由得 | 直接写动作，删除这个词 |
| 仿佛/宛如/好像 | 直接描写，不用比喻 |
| 映入眼帘 | 直接写他看到什么 |
| 心中暗道/暗自思忖 | 直接写想法，不加标签 |
| 沉声道/淡淡地说 | 用动作+内容替代说话标签 |
| 不由自主/情不自禁 | 删除，直接写动作 |
| 嘴角微扬/勾起一抹弧度 | 写具体表情变化（"他笑了一下"） |
| 冷眸/眼底/眸中 | 直接用"他/她"或"眼睛" |

## 二、弱副词禁用表

| 禁用词 | 替换方案 |
|--------|---------|
| 微微点头 | 点头 |
| 淡淡开口 | 他说（或用动作带） |
| 缓缓站起 | 站起来 |
| 轻轻摇头 | 摇头 |
| 略显/稍显 | 删除 |
| 似乎/好像 | 删除 |

**原则：90%的副词删除，剩下的10%替换为具体动作。**

## 三、套话禁用表

| 禁用词 | 替换方案 |
|--------|---------|
| 意义深远/至关重要 | 写出具体影响，不贴标签 |
| 前所未有 | 直接写发生了什么 |
| 堪称/无疑 | 删除 |
| 未来可期/前途无量 | 替换为具体悬念或下一行动 |
| 不难看出/由此可见 | 删除，直接叙述 |
| 于是乎/与此同时/从而/因而 | 删除或改为口语化 |
| 值得注意的是 | 删除 |

## 四、句式禁用规则

| 规则 | 说明 |
|------|------|
| 不用三重排比 | AI喜欢"A，B，C"三件套，压缩为一句话 |
| 不用"不是……而是……" | 拆成两句 |
| 不用"没有……也没有……" | 拆成两句 |
| 不用"虽然……但是……" | 删关联词，拆成两句 |

# 第二层：中级规则（需要针对性改写）

## 五、两遍润色法

### 第一遍：清除AI模式
逐段扫描，按词汇表和句式规则逐一替换。

### 第二遍：AI自审
通读全文，问自己三个问题：

1. "这段哪里还明显是AI写的？"——列出3-5处
2. "这太顺了吗？"——如果是，加个意外
3. "这话像真人说的吗？"——如果是，删解释、加潜台词

## 六、改写范例对照

| AI写法 | 自然写法 |
|--------|---------|
| 她心中五味杂陈，不知道该说什么 | 她张了张嘴，最后只把手里的杯子放回桌上。陶瓷磕出一声脆响 |
| 他是一个很危险的男人 | 他没抬高声音，只是把打火机"啪"地扣上，屋里就安静了 |
| 两人之间的气氛非常微妙 | 她说完那句话后，他没有接，只伸手把她面前的水杯推远了半寸 |
| 一股暖流涌上心头 | 她鼻子一酸，赶紧低头假装系鞋带 |
| 他的眼神深邃而危险 | 他看着她，眼睛黑沉沉的，什么表情都没有 |
| 她不禁想起了往事 | 她想起七年前那个雨夜——那时候她还相信他 |
| 空气中弥漫着说不出的紧张感 | 没有人说话。连呼吸声都变得小心翼翼 |

# 第三层：高级规则（写作时同步应用）

## 七、叙事逻辑层：拒绝"平滑"

AI最大的特点是"顺滑"，起承转合严丝合缝。人类的好故事往往是"疙疙瘩瘩"的。

### 规则1："但是"与"因此"法则（破坏因果链）

| AI写法 | 自然写法 |
|--------|---------|
| A发生了，然后B发生了，最后C发生了 | A发生了，但是B发生了（意外），因此C不得不发生（被迫的选择） |

**操作：永远不要让角色轻易得到想要的东西。**

- ❌ "他拔剑杀死了恶龙。"
- ✅ "他拔剑刺向恶龙，剑却断了，他只能捡起地上的断剑插进龙的眼睛。"

### 规则2：引入"噪音"与"闲笔"
AI习惯：每一句话都在推动剧情，没有废话。

**操作：加入看似无用的细节。**

- 在紧张的谈判前，描写主角鞋子里进了一粒沙子，磨得脚疼
- 在悲伤的葬礼上，描写旁边有人在偷偷看手机里的搞笑视频

**原理：真实的生活充满了干扰项，只有AI才会让全世界都围着主角的情绪转。**

## 八、对话交互层：拒绝"乒乓球"

AI写的对话通常是"你问我答"，高效但极其无聊。

### 规则1："潜台词"原则（话不说满）

| AI写法 | 自然写法 |
|--------|---------|
| "你为什么要背叛我？因为我恨你，而且我想要那笔钱。" | "茶凉了。""我去给你换一杯。""不用换了，反正我也渴够了。" |

**操作：角色永远不直接说出真实意图，用话外之音代替。**

### 规则2：打断与错位
AI习惯：等对方说完完整的句子，再开始说下一句。

**操作：抢话、无视、答非所问。**

- 当A正在深情告白时，B突然问："你刚才是不是踩到我的脚了？"——这种错位感能瞬间打破AI的机械感。

## 九、感官描写层：拒绝"通用形容词"

AI喜欢用宏大的、抽象的形容词（美丽的、恐怖的、令人震惊的）。

### 规则1："生理反应"替代"情绪名词"

| AI写法 | 自然写法 |
|--------|---------|
| 他感到非常恐惧 | 他的膀胱一阵收缩，膝盖骨像是被抽走了 |
| 他很生气 | 他捏碎了手里的玻璃杯，血顺着指缝流下来 |
| 她很难过 | 她鼻子一酸，赶紧低头假装系鞋带 |

**口诀：不要写"他很生气"，要写"他捏碎了杯子"。**

### 规则2：聚焦"局部"而非"整体"

AI习惯：描写一个美女，从头发写到脚趾，面面俱到。

**操作：只抓一个最独特的特征（特写镜头）。**

- ❌ 描写她多美、多优雅
- ✅ 写她眼角的鱼尾纹里夹着一点没卸干净的亮片，在阳光下闪着诡异的光

## 十、人物塑造层：拒绝"完美工具人"

AI创造的角色往往性格单一（好人太好，坏人太坏）。

### 规则1：赋予"怪癖"与"阴暗面"

| AI写法 | 自然写法 |
|--------|---------|
| 侦探是个聪明、正义、冷静的人 | 侦探是个严重的洁癖，每次破案都要戴三层手套 |
| 反派就是纯粹的邪恶 | 反派是个赌鬼，破案的动机是为了赢奖金还债 |

**瑕疵让人物变得真实。**

### 规则2：价值观冲突
**操作：让角色持有某种偏执的、甚至错误的信念。**

- 一个认为"金钱是唯一真理"的父亲，和一个认为"理想高于一切"的儿子。这种不可调和的矛盾是人类戏剧的核心。

## 十一、句式节奏层：拒绝"教科书语法"

AI的句子结构完整，主谓宾齐全，长短句分布均匀。

### 规则1：碎片化表达

| AI写法 | 自然写法 |
|--------|---------|
| 这是一个阳光明媚的早晨，鸟儿在树上歌唱 | 早晨。阳光刺眼。鸟叫个没完，真烦。 |

**操作：模拟大脑的思维跳跃。**

### 规则2：动词为王

| AI写法 | 自然写法 |
|--------|---------|
| 桌子上有一本书 | 一本书躺在桌上，书脊裂了一道口子 |
| 他心里觉得很害怕 | 他腿软了 |

**操作：把静态描述变成动态动作，把"有/是/觉得"换成具体动词。**

# 第十二层：综合自检清单

## 写作完成后逐项检查

1. 有没有「不禁」「仿佛」「不由自主」等高频词？
2. 有没有「微微」「淡淡」「缓缓」等弱副词？
3. 情绪是用动作表现的还是用总结说的？
4. 对白听起来像真人说话吗？（有没有潜台词？有没有打断？）
5. 有没有超过20字的复杂句式？
6. 是否每段都在推进（目标/阻碍/变化）？
7. 有没有连续3句以上结构相似的句子？
8. 环境有没有干扰角色，还是只是背景板？
9. 有没有一次性抛出太多信息？（工作、住址、社交圈——分开放）
10. 角色有没有怪癖或阴暗面？

## 最后的自检三问

写完一段后，问自己三个问题：

1. **"这太顺了吗？"** → 如果是，加个意外，加个阻碍。
2. **"这话像真人说的吗？"** → 如果是，加点潜台词，删掉解释。
3. **"这感觉够具体吗？"** → 如果是，把形容词换成生理反应，把总结换成动作。

---

**记住：AI写的是"正确的废话"，你要写的是"带血的真相"。**
