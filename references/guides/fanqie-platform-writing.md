# 番茄小说平台写作指南

> 以下内容爬取自番茄作家课堂（fanqienovel.com），涵盖新手专区、写作技巧、品类指南三大板块共 131 篇文章，正文汇总约 44 万字。

## 一、开篇法则

### 1.1 黄金开篇三要素

**能让读者点进来 → 能让读者看完开篇 → 能让读者想看后续**

**黄金三秒原则：**
- 从第一个字开始就下钩子
- 三段话以内必有重点
- 300字之内必进主题
- 不要在开篇做背景介绍/旁白/故事框架说明

**切入点要求：快、狠、准**
- 前置高光事件：把冲突/爽点/虐点/猎奇点最明显的事件前置
- 前置人设特性事件：用一个具体事件证明人设，不靠旁白
- 保留冲击性和故事性，同时留有悬念
- 每个字尽可能保证功能性，避免"废话"

### 1.2 开篇剧情设计

写完开篇后自检：
1. 是否勾人？
2. 是否有悬念？
3. 是否有情绪拉扯？
4. 是否快节奏？

**铁律：** 不要枯燥。只要有一点点平淡的地方就要改，哪怕只是"一点点"。

### 1.3 八种有效开篇方式

| 方式 | 说明 |
|------|------|
| 冲突前置 | 把最强冲突放在开头 |
| 悬念钩子 | 用一个未解之谜开场 |
| 身份反差 | 主角身份自带冲突感 |
| 倒计时开场 | 用时间限制制造紧迫感 |
| 高光前置 | 把名场面/转折点提前 |
| 对话开场 | 一句有张力的对白直接开始 |
| 违背常理 | 一个无法解释的现象开场 |
| 情感冲击 | 强烈的情绪瞬间抓住读者 |

## 二、节奏控制

### 2.1 三种节奏问题及改进

**节奏拖沓：** 故事总是不进入主线，前期描写对剧情无推动作用
- → 删减不必要内容，从冲突入手，围绕主线和关键情节展开
- → 每个章节设定明确目标，避免偏离主线

**节奏混乱：** 情节发展毫无逻辑，场景转换突兀
- → 通过人物的行动、心理变化进行衔接
- → 提前梳理核心规则清单，避免规则矛盾

**情节前后矛盾：** 世界规则/人物能力在不同情节不一致
- → 写作前明确世界法则、能力限制，关键情节对照清单

### 2.2 过渡章处理（核心）

**关键认知：不存在过渡章。**
- 过渡章不是"故事A结束到故事B开端"的中间地带
- 过渡章 = 制造下一个期待感的铺垫过程
- 要在过渡中制造对故事B的**期待感、危机感、信息差**
- 一定不要让读者在故事推进中失去期待感

**过渡章写法：**
- 加快节奏，缩短平淡剧情
- 写一些有魅力的副CP吸引读者
- 写"结算时刻"——大剧情后的物质/情感变化、配角反应
- 如果觉得故事没劲 → 大概率铺垫少了，往上加配角、加支线（但必须为主角服务）

### 2.3 爽点密度设计

**常见错误：**
- 爽点类型单一（全程只围绕"打脸"）
- 爽点突兀（缺乏合理逻辑支撑与情感积累）
- 爽点延迟（铺垫太久，读者失去兴趣）

**多样化爽点类型：**
| 类型 | 说明 |
|------|------|
| 打脸爽点 | 主角反击反派，证明自己 |
| 升级爽点 | 修为/能力/地位提升 |
| 智斗爽点 | 通过计谋破解阴谋 |
| 逆袭爽点 | 从底层成长为强者 |
| 情感满足 | 修成正果、深厚情谊 |
| 信息差爽点 | 读者知道但角色不知道，期待"掉马" |

**爽点排布铁律：**
- 连续2章平淡 → 第3章必须有爆点
- 每章至少一个小爽点/信息增量
- 爽点之间要有层次，避免集中堆砌或长时间空缺

## 三、冲突设计

### 3.1 冲突乏力三大表现

1. **平淡无冲突：** 主角行动没有阻碍，故事过于顺利
2. **高潮无铺垫：** 最终冲突突然出现，无线索/伏笔/情绪积累
3. **高潮无后续：** 高潮结束后无后续影响，沦为"孤立事件"

### 3.2 冲突改进方法

**强化冲突的根本：双方都有"不能退让的理由"**
- 给双方加上"我必须这么做"的动机
- 问自己：如果主角不这么做，会失去什么？
- 代价越具象，读者越理解

**高潮前增加伏笔：** 在高潮前3-5章埋设线索，通过"线索递进"引导读者期待

**增加高潮后剧情设计：** 高潮后1-2章描写高潮带来的改变（主线推进、人设成长、世界关系变化）

### 3.3 转折生硬的根源

**视角问题：** 作者知道后续要发生什么，但读者不知道
- → 适当加入内心独白，为主角行为做预期解释
- → 不要让读者觉得"主角行为不合理"

## 四、人物塑造

### 4.1 让人物有"活人感"四步法

**① 填过往** — 用回忆、伏笔让人物的行为有逻辑、有温度
- 没有无缘无故的善恶，每个人物的言行源于过往经历
- 过往不能一次性全盘托出，要穿插在情节中一点一点放出来
- ❌ 不要加对剧情推动无意义的回忆

**② 加细节** — 用动作、神态、心理的刻画让情绪有画面
- 动作微表情、神态变化远比直白的"他很生气"更有力量
- ❌ 不必每句话都加动作，必须是对剧情有意义的才加

**③ 添成长** — 让角色随剧情发展产生合理变化
- 成长核心逻辑：困境/冲突 → 挣扎/反思 → 改变/突破 → 沉淀/蜕变
- 避免"无理由成长""一蹴而就成长"

**④ 增复杂** — 避免人物脸谱化，增加"活人感"
- 好人也有缺点，坏人也有苦衷
- 配角不能工具化

### 4.2 成长类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 主角成长（核心） | 贯穿主线 | 懦弱→勇敢、自私→担当、迷茫→坚定 |
| 配角成长（辅助） | 衬托主角，推动主线 | 冲动→可靠、任性→悔悟 |
| 反派成长（反向） | 行为更有逻辑 | 偏执→醒悟、被操控→找回自我 |

### 4.3 CP塑造与人设崩塌

**CP塑造要点：**
- 双方要有独立的人物弧线
- 感情发展要有合理的递进层次
- 避免"一见钟情"式的无理由相爱

**避免人设崩塌：**
- 建立人物行为准则清单，关键情节对照
- 角色在压力下的选择必须符合其核心性格
- 成长不等于推翻原有设定

## 五、章节工程

### 5.1 好章节的标准

- [ ] 每章有明确目标，不偏离主线
- [ ] 有至少一个冲突推进或信息增量
- [ ] 有情绪变化（不能从头平到尾）
- [ ] 章末设置了钩子/悬念
- [ ] 没有大段无关的环境描写
- [ ] 没有大段设定说明

### 5.2 高效改文方法

**数据差（读完率低、跳章多）的原因：**
- 有毒点 → 删掉或替换争议内容
- 过渡太水 → 加快节奏，缩短平淡期
- 期待感断裂 → 检查是否失去了读者的期待

**单章修改方向：**
- 节奏快慢调整：快节奏章用短句多动作，慢节奏章用描写内心
- 冲突不够 → 加对抗理由、加"必须这么做"的动机
- 转折生硬 → 加内心独白做预期解释

### 5.3 拒绝流水账

- 删减不必要内容，围绕主线展开
- 每个事件要改变人物情绪/状态
- 不要写"起床→刷牙→吃饭"式的日常流水
- 直接切入核心事件

## 六、大纲设计

### 6.1 大纲怎么写才不崩

**核心原则：**
- 主线清晰：主角要做什么？为什么做？阻碍是什么？
- 阶段性目标：每个阶段有明确的小目标
- 伏笔管理：每个伏笔预设回收章节

**感情线与事业线双线并行：**
- 两条线交替推进，互不干扰但互相影响
- 感情线推进时机：事业线处于低潮/缓冲期时
- 事业线高潮前感情线埋下伏笔

### 6.2 开书前的筹备

1. 题材选择：分析市场热度，找到差异化切入点
2. 设定新颖：在流行框架上做微创新，避免完全雷同
3. 开篇精铸：打磨前300字、前1500字
4. 大纲完整：不少于20章的大纲骨架
5. 心态准备：接受前期数据波动

## 七、品类与题材

### 7.1 拉新向写作核心

拉新文核心要求：**直白、吸睛、快节奏**

**题材五大特点：**
1. 开篇抓眼：3秒内抛出核心冲突或悬念
2. 节奏要快：爽点或情绪爆点密集
3. 语言直白简洁：短句推进，避免复杂词汇
4. 代入感要强：擅用大众化话题，感官描写增强代入
5. 人设鲜明极端：性格特点突出，一眼记住

### 7.2 常见男频品类要点

| 品类 | 核心要点 |
|------|---------|
| 玄幻 | 世界架构+境界体系+奇遇成长 |
| 都市 | 贴近现实+金手指反差+爽点密集 |
| 科幻 | 科技体系自洽+想象力+人文思考 |
| 历史 | 史实严谨+合理演义+权谋智斗 |
| 悬疑推理 | 逻辑自洽+伏笔回收+节奏紧张 |
| 游戏 | 系统设定清晰+升级爽感+竞技对抗 |

### 7.3 常见女频品类要点

| 品类 | 核心要点 |
|------|---------|
| 古代言情 | 人设优先+情感递进+宫斗宅斗 |
| 现代言情 | 现实议题+情感拉扯+独立成长 |
| 脑洞文 | 创意为王+设定新颖+反转不断 |
| 年代文 | 时代感真实+逆袭爽感+温馨日常 |

### 7.4 改编风向

- **打脸逆袭甜宠**仍是改编三大核心要素
- **短剧改编**偏好：快节奏、强冲突、人设极端
- **IP改编**看重：世界观完整、人物弧线清晰、情感共鸣强

## 八、新手避雷

| 坑点 | 正确做法 |
|------|---------|
| 第一章出场过多人物 | 只保留主角+反派+贵人（≤5人） |
| 开局写背景设定 | 先给金手指/冲突，设定揉进剧情 |
| 流水账式开头 | 直接切入核心事件 |
| 心理描写过长 | 前几章少心理活动，用行动表现 |
| 过渡章平淡 | 不存在过渡章，每章都要有期待感 |
| 冲突无铺垫 | 前3-5章埋线索，引导读者期待 |
| 高潮无后续 | 高潮后写影响力，让情绪延续 |
| 爽点类型单一 | 打脸/升级/智斗/逆袭/情感多种搭配 |
| 规则前后矛盾 | 建立规则清单，关键情节对照 |
| 人物行为无逻辑 | 给人物加过往，行为源于经历 |

## 九、写作心态

- **接受数据波动：** 换地图/换副本时掉追读是正常的
- **强迫症式改文：** 任何觉得平淡的地方就改，不要有恃无恐
- **读者是随时会跑的客户：** 松懈一句话就流失一批读者
- **长期主义：** 好故事是改出来的，不是写出来的
