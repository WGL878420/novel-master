# 去AI味指南

整合自 leenbj 的7类检测 + Humanizer-zh 的24模式 + Tomsawyerhu 的去AI味原则

## 核心原则
**能用动作不用总结，能用对白不用解释，能写具体不写抽象**

## AI味七类检测

### 1. 高频AI词汇
| 禁用词 | 替换方案 |
|--------|---------|
| 不禁 | 直接写动作，删除 |
| 仿佛/宛如 | 直接描写，不用比喻 |
| 映入眼帘 | 直接写他看到什么 |
| 心中暗道/暗自思忖 | 直接写想法 |
| 沉声道/淡淡地说 | 用动作+内容替代说话标签 |
| 不由自主/情不自禁 | 删除，直接写动作 |
| 嘴角微扬/勾起一抹弧度 | 写具体表情变化 |
| 冷眸/眼底/眸中 | 直接用"他/她" |

### 2. 弱副词滥用
> 微微点头 / 淡淡开口 / 缓缓站起 / 轻轻摇头

**改法**：删除90%，剩下的替换为具体动作。

### 3. 意义膨胀
> 意义深远 / 前所未有 / 堪称 / 无疑 / 至关重要

**改法**：删除标签，写出具体影响。

### 4. 套话结语
> 未来可期 / 前途无量 / 充满希望

**改法**：替换为具体悬念、未解决冲突或下一行动。

### 5. 论文腔结构
> 不难看出 / 由此可见 / 值得注意的是

**改法**：删除，直接叙述。

### 6. 书面腔
> 于是乎 / 与此同时 / 从而 / 因而

**改法**：删除或改为口语化表达。

### 7. 三重排比
AI 习惯把东西排成三个一组。

**改法**：压缩为一句话。

## 两遍润色法

### 第一遍：清除AI模式
逐段扫描，按上述7类逐一替换。

### 第二遍：AI自审
通读全文，问自己："这段哪里还明显是AI写的？"
列出3-5处问题，逐处改写。

## 改写范例对照

| 原版(AI味) | 改写(自然) |
|------------|-----------|
| 她心中五味杂陈，一时间不知道该说什么 | 她张了张嘴，最后只把手里的杯子放回桌上。陶瓷磕出一声脆响 |
| 他是一个很危险的男人 | 他没抬高声音，只是把打火机"啪"地扣上，屋里就安静了 |
| 两人之间的气氛非常微妙 | 她说完那句话后，他没有接，只伸手把她面前的水杯推远了半寸 |
| 一股暖流涌上心头 | 她鼻子一酸，赶紧低头假装系鞋带 |
| 他的眼神深邃而危险 | 他看着她，眼睛黑沉沉的，什么表情都没有 |
| 她不禁想起了往事 | 她想起七年前那个雨夜——那时候她还相信他 |
| 空气中弥漫着一种说不出的紧张感 | 没有人说话。连呼吸声都变得小心翼翼 |

## 检查清单
写作完成后逐项检查：
- [ ] 有没有「不禁」「仿佛」「不由自主」等高频词？
- [ ] 有没有「微微」「淡淡」「缓缓」等弱副词？
- [ ] 情绪是用动作表现的还是用总结说的？
- [ ] 对白听起来像真人说话吗？
- [ ] 有没有超过20字的复杂句式？
- [ ] 是否每段都在推进（目标/阻碍/变化）？
- [ ] 有没有连续3句以上结构相似的句子？

## 写前预防
去AI味不是最后润色，而是从写作时就开始的约束：
1. 写动作，不写感受
2. 写对话，不写解释
3. 写具体细节，不写抽象描述
4. 每人说话带身份感
5. 段落长短交错，句式变化

### 进阶预防：高阶AI味自查

**1. 打破"一问一答式"对话**
不要让你的对话变成纯信息交换。现实中没人会直接回答问题。
- ❌ "铜钱上有什么发现？" "没有指纹，被擦干净了。"
- ✅ 先甩情绪，再给信息。"该死的，那几枚铜钱被磨得像新的一样，连个皮屑都找不到。"

**2. 破坏"完美逻辑链"**
不要让线索被完美地逐个发现。现实探案是碎片的、混乱的。
- ❌ "第一起在城北，第二起在开发区，第三起在桥洞。所以凶手在画圆。"
- ✅ 先怀疑错方向，再被意外推翻，最后才灵光一闪。

**3. 环境要"干扰"角色，不只是"背景板"**
别让环境只当装饰——让它影响角色的行动和情绪。
- ❌ "窗外是车水马龙。两百万人，谁也不知道谁会变成下一个目标。"
- ✅ "窗外的喇叭声吵得林深太阳穴突突直跳。他伸手关了窗，世界安静了，但他的脑子没安静。"

**4. 推理要笨拙，不要顺滑**
让主角猜错、走弯路、犹豫。
- ❌ "手机被凶手拿走了。这是对抗，他知道我们会查手机。"
- ✅ "手机不见了……被拿走了？不对，也许是他自己关机扔了？等等——凶手拿走的。凶手不想让我们看到里面的东西。"

**5. 多个信息分开给，不要堆在一起**
不要一次性抛出"工作、住址、社交圈、银行流水"这种清单。
- ✅ 先给最重要的一个。等角色消化了，再给下一个。中间插动作、插情绪、插干扰。

**6. 对话要有"潜台词"和"无效内容"**
现实中的人说话不直奔主题。
- ✅ 回答前先叹气、抱怨、骂一句。用动作代替说话标签。
- 林深没抬头："红绳呢？"
- 陈敏靠在椅背上，揉了揉太阳穴："淘宝爆款，九块九包邮。"

## 五层去AI味写作技法

整合自深层写作规则：叙事逻辑层、对话交互层、感官描写层、人物塑造层、句式节奏层。

### 第一层：叙事逻辑层——拒绝"平滑"

AI生成的故事最大的特点是"顺滑"，起承转合严丝合缝。人类的好故事往往是"疙疙瘩瘩"的。

#### "但是"与"因此"法则（破坏因果链）

- **AI习惯**：A发生了，然后B发生了，最后C发生了。（线性流水账）
- **去AI规则**：A发生了，**但是**B发生了（意外），**因此**C不得不发生（被迫的选择）。
- **操作**：永远不要让角色轻易得到想要的东西。如果AI写"他拔剑杀死了恶龙"，你要改成"他拔剑刺向恶龙，剑却断了，他只能捡起地上的断剑插进龙的眼睛"。

#### 引入"噪音"与"闲笔"

- **AI习惯**：每一句话都在推动剧情，没有废话。
- **去AI规则**：加入看似无用的细节。
- **操作**：
  - 在紧张的谈判前，描写主角鞋子里进了一粒沙子，磨得脚疼。
  - 在悲伤的葬礼上，描写旁边有人在偷偷看手机里的搞笑视频。
- **原理**：真实的生活充满了干扰项，只有AI才会让全世界都围着主角的情绪转。

### 第二层：对话交互层——拒绝"乒乓球"

AI写的对话通常是"你问我答"或"信息交换站"，非常高效但极其无聊。

#### "潜台词"原则（话不说满）

- **AI习惯**：
  - "你为什么要背叛我？"
  - "因为我恨你，而且我想要那笔钱。"
- **去AI规则**：角色永远不直接说出真实意图。
- **操作**：
  - "茶凉了。"
  - "我去给你换一杯。"
  - "不用换了，反正我也渴够了。"

#### 打断与错位

- **AI习惯**：等对方说完完整的句子，再开始说下一句。
- **去AI规则**：抢话、无视、答非所问。
- **操作**：当A正在深情告白时，B突然问："你刚才是不是踩到我的脚了？"这种错位感能瞬间打破AI的机械感。

### 第三层：感官描写层——拒绝"通用形容词"

AI非常喜欢用宏大的、抽象的形容词（如：美丽的、恐怖的、令人震惊的）。

#### "生理反应"替代"情绪名词"

- **AI习惯**：他感到非常恐惧。
- **去AI规则**：描写身体的失控。
- **操作**：他的膀胱一阵收缩，膝盖骨像是被抽走了，喉咙里发出的声音像是指甲划过黑板。
- **口诀**：不要写"他很生气"，要写"他捏碎了手里的玻璃杯，血顺着指缝流下来"。

#### 聚焦"局部"而非"整体"

- **AI习惯**：描写一个美女，会从头发写到脚趾，面面俱到。
- **去AI规则**：只抓一个最独特的特征（特写镜头）。
- **操作**：别写她多美，写她眼角的鱼尾纹里夹着一点没卸干净的亮片，在阳光下闪着诡异的光。

### 第四层：人物塑造层——拒绝"完美工具人"

AI创造的角色往往服务于剧情，性格单一（好人太好，坏人太坏）。

#### 赋予"怪癖"与"阴暗面"

- **AI习惯**：侦探是个聪明、正义、冷静的人。
- **去AI规则**：给他加点毛病。
- **操作**：这个侦探是个严重的洁癖，每次破案都要戴三层手套；或者他是个赌鬼，破案的动机是为了赢奖金还债。瑕疵让人物变得真实。

#### 价值观冲突

- **AI习惯**：大家都认同普世价值。
- **去AI规则**：让角色持有某种偏执的、甚至错误的信念。
- **操作**：一个认为"金钱是唯一真理"的父亲，和一个认为"理想高于一切"的儿子。这种不可调和的矛盾是人类戏剧的核心，AI很难写出这种深刻的撕裂感。

### 第五层：句式节奏层——拒绝"教科书语法"

AI的句子结构通常很完整，主谓宾齐全，长短句分布均匀。

#### 碎片化表达

- **AI习惯**：这是一个阳光明媚的早晨，鸟儿在树上歌唱。
- **去AI规则**：模拟大脑的思维跳跃。
- **操作**：早晨。阳光刺眼。鸟叫个没完，真烦。

#### 动词为王

- **AI习惯**：大量使用"是"、"有"、"觉得"等弱动词。
- **去AI规则**：把静态描述变成动态动作。
- **操作**：
  - AI：桌子上有一本书。
  - 人：一本书躺在桌上，书脊裂了一道口子。

## 速查：AI vs 自然对照表
| 场景 | AI写法 | 自然写法 |
|------|--------|---------|
| 惊讶 | 他震惊地瞪大了眼睛 | 他手里的筷子掉在了桌上 |
| 愤怒 | 一股怒火涌上心头 | 她把杯子重重磕在桌上，水溅了出来 |
| 悲伤 | 她的心中充满了悲伤 | 她没说话，只是把脸埋在掌心里 |
| 紧张 | 气氛变得紧张起来 | 她数着自己的心跳。一下，两下，三下 |
| 喜欢 | 他的心里产生了一种异样的感觉 | 他不自觉地多看了她两眼 |

## 自检三问

写完一段后，问自己三个问题：

1. **这太顺了吗？** 如果是，加个意外，加个阻碍。
2. **这话像真人说的吗？** 如果是，加点潜台词，删掉解释。
3. **这感觉够痛吗？** 如果是，把形容词换成生理反应。

记住：AI写的是"正确的废话"，你要写的是"带血的真相"。
