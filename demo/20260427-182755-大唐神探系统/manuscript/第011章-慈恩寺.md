# 第十一章 慈恩寺

慈恩寺在长安城东南八里外，早就断了香火。残破的院墙在杂草丛中若隐若现，山门上的匾额歪了一半，上面的字迹被风雨侵蚀得只剩最后一笔。苏尘小时候在历史书上看过关于这座寺庙的记载——贞观初年曾经香火鼎盛，后来一场大火烧了大半，一直没有重修，渐渐就荒废了。附近的人都说这里闹鬼，很少有人靠近。

苏尘到的时候已经是下午。日头偏西，寺庙的阴影拉得很长，荒草在风里摇晃，发出沙沙的声响。他没有从正门进——正门那条路太明显，如果有人跟踪一眼就能看到。他绕到寺庙后面，从一段塌了大半的院墙翻进去，落地的时候踩碎了一块瓦片，清脆的碎裂声在空旷的寺院里格外刺耳。他蹲下来等了片刻，确认没有人才站起来。

寺里比他想象的要大。正殿的屋顶已经塌了，露出里面残破的佛像，半边脸被烟熏得发黑，另半边脸在斜阳下呈现出一种诡异的慈悲。偏殿只剩下几根立柱，荒草从砖缝里疯长出来，最高的已经半人高了。后院更荒凉，几间僧房的屋顶都塌了，只剩墙基还隐隐可辨。

苏尘站在后院中央环顾了一圈。如果柳氏的尸体被埋在这里，埋的人一定会选一个不容易被找到的位置——但又不能太随意，因为埋尸的人迟早要回来确认。他蹲下来观察地面的状态。十五年的荒草生长会把地面覆盖得很均匀，但被人翻动过的泥土，植被分布一定不一样。

他从后院的墙角开始一米一米地排查。正殿周围的地面、偏殿的废墟旁边、走廊的砖缝之间——每走几步他就蹲下来用手按一按泥土的硬度，观察草的生长方向。杂草不会撒谎。被翻过的泥土即使过了十五年，草根扎下去的方向和密度也会和原生土不一样。

排查到正殿的佛像底座前面时，他停住了。

这座佛像已经倒塌了半边，底座用青砖砌成，大约半人高。底座周围的野草比别处矮了一截——不是长得不好，是因为下面的土是后来翻过再回填的，草籽落到上面的时间比周围晚，根扎得也没那么深。

【洞察之眼已激活】
【线索：佛像底座下方泥土颜色异常——约十五年前翻动回填痕迹，面积约一人身长】

苏尘蹲下来，用手拨开表层的浮土。下面的泥土果然松软——和周围压实的原土手感完全不一样。有人在这里挖过坑，把什么东西放进去，又把土填了回来。时间长雨水冲刷，表面已经看不出坑的形状，但一挖就露馅了。

他没有工具。短刀沿着底座的边缘切下去，一格一格地挖。泥土夹杂着小碎石，挖起来并不费力——当年填土的人没有把土夯实，而是有意让它保持疏松，可能是为了方便以后再挖开。这个发现让苏尘的心沉了一下：埋尸的人知道自己迟早还要回来。

挖到大约两尺深的时候，刀尖碰到了硬物。不是石头——刀尖碰上去的声音不对，是一种钝而实的回响。他换了个方向继续挖，把周围的浮土一层一层扒开。

露出来的是一块木板，已经腐朽了，边缘发黑，但还能看出棺材板的形状。木料是最便宜的松木，薄薄一层，连漆都没上。这种薄皮棺材在长安城外的义庄里几文钱就能买一口，埋了也不会有人注意。

苏尘沿着棺材的边缘把浮土全部清理干净。整具棺材完全露了出来——不大，很薄，简陋到近乎寒酸。但就是这口寒酸的薄皮棺材，装着一个让整个长安城暗流涌动了十五年的秘密。

他用刀尖撬开棺材盖。木板已经朽了，稍微用力就裂开了。

棺材里面不是白骨。

是一具穿戴整齐的女尸，衣物完好，发髻还在。脸上的皮肉已经风干了，呈现出深褐色的状态，但整体轮廓还能看出生前的面容——柳眉，薄唇，面容清秀。不是白骨化的状态。这具尸体在入棺之前经过处理——有人用石灰和香料做过防腐。所以十五年后，她依然保持着下葬时的姿态。

柳氏。

苏尘蹲在棺材旁边屏住呼吸。她的双手交叠放在腹部，手指修长，指甲上还残留着淡红色的蔻丹痕迹。衣物是宫里才有的料子，已经褪色发脆。纹路依然清晰——云纹暗花，是贞观年间宫中贵人常穿的样式。致命伤在胸口——一柄短刀插在心口位置，刀柄已经朽断了，但刀身还深深嵌在肋骨之间。

不是暴病而亡。是谋杀。

但真正让苏尘在意的是另一件事——尸体的摆放姿态。柳氏的双手叠放得很整齐，衣襟被抚平过，头发被仔细梳理过，连发髻上的银钗都端正地插在原位。杀她的人没有像处理一件垃圾一样把她扔进坑里，而是像对待一具需要体面下葬的尸体一样，帮她整理好了仪容。

这和连环灭口的冷酷手法完全不同。杀王世充手下那些人的凶手干脆利落——毒杀、刺杀、一刀毙命。但杀柳氏的人，在杀了她之后，还为她整理了遗容。凶手认识柳氏。甚至可能在乎她。

苏尘的目光落在柳氏交叠的双手上。右手手腕上戴着一只玉镯。玉质温润，在昏暗的光线下泛着柔和的色泽——不是新玉，是经过岁月摩挲的老玉。在这种地方十五年的风吹雨打，玉镯依然完整无损。

他小心地把玉镯从手腕上取下来。死者的皮肉已经干缩，取下来并不费力。玉镯内侧刻着两个字。字很小，他凑到光线底下仔细辨认：

“念安。”

不是名字。是一句祝福，又是恋人之间的信物。刻痕不是新雕的——玉镯的包浆已经把字的边缘磨得圆润了，说明在柳氏生前，这只镯子已经被佩戴了很久。

苏尘把玉镯贴身收好，然后把棺材盖重新盖上，把泥土回填，尽量恢复了原状。他一个人搬不动一具棺材，不可能把尸体带回去。但他可以把魏征带到这里来亲眼确认——只要魏征看到柳氏的尸体和胸口的刀伤，他就不会再有任何犹豫了。

回城的路上夕阳把长安城的城墙染成暗金色。苏尘走得不快不慢，脑子里一直在想那只玉镯。念安。这个词不像人名字——更是一种祝愿或者嘱托。什么人会在玉镯上刻这样的字送给柳氏？

系统提示忽然在视野中弹出。

【主线任务进度更新】
【第一幕·长安暗流 → 第二幕·慈恩谜影】
【关键物证：柳氏遗体确认——非暴病，系他杀（胸口刀伤）】
【新线索物品：玉镯（刻字“念安”）——推测为重要信物】
【新技能解锁：逻辑推演（初级）——可将已知线索进行关联分析，发现隐藏逻辑链】
【下一阶段目标：将信息告知魏征，确认同盟】

苏尘停住脚步查看了一下新技能。逻辑推演（初级）可以在他整理线索时自动提示关键信息之间的隐藏关联。他试了一下——把口袋里的证物在脑海中过一次，技能立刻给出了一组关联提示。

【逻辑推演：柳氏尸体经防腐处理 → 凶手具备防腐知识 → 非普通杀手 → 可能为宫中御医或仵作】
【逻辑推演：玉镯刻字“念安” → 与柳氏亲密关系者所赠 → 凶手可能并非幕后主使本人，而是受指使的执行者】
【逻辑推演：幕后主使选择慈恩寺埋尸 → 该地点与主使存在隐秘关联】

三条推演结果排在一起。苏尘一条一条看完，在心里把线索又理了一遍。杀人者具备专业知识，可能是个御医或仵作。他和柳氏有亲密关系，甚至是受了她的嘱托——这解释为什么尸体被整理过仪容。而慈恩寺这个地点，和幕后主使之间存在着某种还没有浮出水面的关联。

他加快脚步往城门方向走。天快黑了，他必须在城门关闭之前回去。柳氏的尸体找到了，但接下来要做的事比找尸体更危险——他要把结果告诉魏征。而魏征一旦确认了尸体，就意味着正式向那个幕后主使摊牌。

长安城的城门在他身后缓缓合拢。夕阳落尽，城墙上亮起了第一排火把。

苏尘走进城门时，一个意料之外的人正在等他。

王直。

大理寺少卿王直站在城门内侧的阴影里，穿着一身半旧便服，专门在这里等了很久。看到苏尘进来他快步迎上来，脸色很凝重。

“你这一天去哪了？”

苏尘看着他紧皱的眉头，心里升起一股不好的预感：“出什么事了？”

“出大事了。”王直压低声音，目光迅速在四周扫了一圈，确认没有人在注意他们，“今天下午宫里传出消息——王世充死了。”

苏尘站在原地，像被人从头浇了一盆冰水。他知道王世充被召进宫凶多吉少，但没想到会这么快。上午召进去，下午就死了。效率之高、下手之快，说明那个人根本就没有打算让王世充活着走出皇宫。

“怎么死的？”

“宫里说是突发急症，在偏殿里倒下的。太医赶到的时候人已经没了。”王直的声音压得更低了，“宫里的说法是操劳过度——但我不信。一个人早上还好好的，下午就急症死了？没这么巧。”

苏尘没有说话。王世充死了。这意味着幕后主使已经开始收网了。王世充是第一个——接下来就是他手下的所有人，包括名单上的每一个人。

而名单上最后一个人，是苏尘。

“少卿大人，你现在立刻回大理寺，就当今晚没见过我。”

王直盯着他：“你要去做什么？”

“去找一个人。如果我能活着回来，这案子就破了。如果我回不来——”苏尘顿了一下，“替我照顾好李薇。”

他没有等王直回答，转身走进了暮色中的长安城。

王直站在原地，看着他消失在巷口拐角处。一个见习捕快，手里攥着扳倒半个朝堂的证据，在偌大的长安城里和藏在宫里的那个人赛跑。而他唯一的盟友是一个很久不上朝的宰相，和一个死去的妃子留在玉镯上的两个字。

大理寺见习捕快苏尘这个名字，如今已经摆在御案上了。
